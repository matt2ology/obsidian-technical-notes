When you create a new app Django will set you up with default files in the app's folder

```text
└───website                 # <- Parent project folder
    │   db.sqlite3
    │   manage.py
    │
    ├───music               # <- App created `python manage.py startapp music`
    │   │   admin.py
    │   │   apps.py
    │   │   models.py
    │   │   tests.py
    │   │   views.py
    │   │   __init__.py
    │   │
    │   └───migrations
    │           __init__.py
    │
    └───website
```
